# 鎌倉・室町学習ひろば（骨組み）

小学校6年生向けに、鎌倉時代・室町時代の学習をサポートする単一HTMLアプリの骨組みを用意しています。オフラインでも動作し、後からコンテンツ（本文・図版・問題）を差し替えできる構成です。

## ファイル構成

- `index.html` : HTML/CSS/JavaScript を同梱した学習サイト本体。ブラウザで直接開けます。
- `assets/` : 図版や追加素材を配置するディレクトリ（現在は空）。
- `README.md` : 本ファイル。先生方向けの使い方ガイド。
- `AGENTS.md` : 開発メモ。ビルドやテストは不要です。
- `LICENSE` : MIT License。

## 授業の使い方（15〜20分）

1. **導入（3分）** : プロジェクターで `index.html` を提示し、タブ構成（時代の流れ／重要用語／人物／練習問題／振り返り）を説明します。先生モードをONにして、採点結果が確認できることを示します。
2. **共同確認（7分）** : 「時代の流れ」タブで、鎌倉・南北朝・室町の出来事を簡単に確認。フィルター操作やキーワード検索の使い方を練習します。「人物」タブでクイズを1問出題して、全体の理解度を見ます。
3. **個別学習（5〜7分）** : 児童に各自の端末でタブを切り替えながら情報を調べてもらいます。重要用語カードの反転操作や、振り返りメモの使い方を体験させてください。
4. **まとめ（3分）** : 「練習問題」タブで10問セットを解かせ、採点結果を全体で共有します。誤答のフィードバックを確認しながら、振り返りメモに学びと疑問を入力させます。

## 家庭学習の使い方（10分）

1. **準備（1分）** : 端末で `index.html` を開き、最後に学習したタブから再開できます。
2. **復習（6分）** : 「重要用語」カードで語句を確認しつつ、わからない語は検索で絞り込みます。続けて「人物」タブのクイズを1〜2問解いて、理解を確かめます。
3. **仕上げ（3分）** : 「練習問題」タブで短時間の確認テスト（選択式／記述式）を実施。結果と誤答のメモを見ながら「振り返り」タブで学んだことを更新し、次回授業に向けた質問を整理します。

## 開発メモ

- データは `index.html` 最下部の `const DATA` にまとまっています。JSON相当の配列を差し替えるだけで内容を更新できます。
- ローカルストレージに保存される情報は、先生モードの状態・練習問題の進捗・振り返りメモのみです。個人を特定するデータは保存しません。
- 依存ライブラリは使用していません。ファイルを配布するだけでそのまま利用できます。
